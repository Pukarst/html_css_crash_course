-<ul> is for the unordered list like bullets and <ol> is for the ordered list like numbers,alphabet.
-<table> organized the tabular data with <tr> for rows, <th> for header cell and <td> for the data cells.
-wrapping in a horizontal scrollable container (overflow-x:auto) keeps tables usable on the small screens.


-links connect the site together.
-target="_blank" opens the link in the new tab.
